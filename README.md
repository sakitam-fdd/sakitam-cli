## sakitam-cli

> 一个项目生成的脚手架工具。

## Quickstart

```bash
npm install -g sakitam-cli
# or
yarn global add sakitam-cli

// list available templete
sakitam list
// generate project
sakitam init tpl-base project
cd project
npm i
npm run dev
```

## Development

```bash
git clone https://github.com/sakitam-fdd/sakitam-cli.git
cd sakitam-cli
npm link
sakitam -h
```
